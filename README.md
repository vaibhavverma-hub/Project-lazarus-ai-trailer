# 🎬 Project Lazarus — AI Movie Trailer Generator

Transform a simple movie idea into a cinematic Hollywood-style trailer using Generative AI.

Project Lazarus is an end-to-end AI pipeline that automatically generates a movie concept, screenplay, storyboard, visual prompts, AI-generated visuals, voice narration, and a final cinematic trailer.

---

## 🚀 Features

- 🎥 AI-powered movie concept generation
- 📝 Automatic screenplay generation
- 🎭 Character generation and consistency
- 🎬 Storyboard & shot planning
- 🖼️ Cinematic image prompt generation
- 🎨 AI-generated movie stills
- 🎙️ AI voice narration
- 🎞️ Automated trailer creation

# 🎬 Project Lazarus — AI Movie Trailer Generator

Transform a simple movie idea into a cinematic Hollywood-style trailer using Generative AI.

Project Lazarus is an end-to-end AI pipeline that automatically generates a movie concept, screenplay, storyboard, visual prompts, AI-generated visuals, voice narration, and a final cinematic trailer.

---

## 🚀 Features

- 🎥 AI-powered movie concept generation
- 📝 Automatic screenplay generation
- 🎭 Character generation and consistency
- 🎬 Storyboard & shot planning
- 🖼️ Cinematic image prompt generation
- 🎨 AI-generated movie stills
- 🎙️ AI voice narration
- 🎞️ Automated trailer creation

## ⚙️ How It Works

The pipeline follows a structured AI workflow:

1. The user provides a movie title and genre.
2. Gemini generates an original movie concept.
3. The concept is expanded into a cinematic screenplay.
4. A storyboard is generated, breaking the trailer into visual scenes.
5. Each scene is translated into detailed cinematic image prompts.
6. AI-generated movie stills are created from those prompts.
7. A professional voice-over narrates the trailer.
8. The visuals, narration, and music are assembled into the final cinematic trailer.

## ▶️ How to Run

1. Clone the repository.
2. Open the notebook in Google Colab.
3. Add your Google Gemini API key.
4. Run the notebook cells sequentially.
5. Generate the movie concept, screenplay, storyboard, and visual prompts.
6. Generate the trailer assets and assemble the final trailer.
